# Veritas

Veritas is a flash card website. It allows users to create, practice and update flash card sets. 

## Usage

### Normally

> [!WARNING] 
> This section will be updated once the github pages deployment is online.

### Setting up a development server

Prerequisites:
- git is required if you intend to clone the repository directly. If you instead download it as a zip file, you will require a program to unzip the archive.
- [Node.js and NPM](https://nodejs.org/en/download) must be installed on the machine.

```sh
# Clone the repository and move into it
git clone https://github.com/michael-m-2983/veritas.git && cd veritas

# Install project dependencies
npm install

# Run a development server
npm run dev
```
